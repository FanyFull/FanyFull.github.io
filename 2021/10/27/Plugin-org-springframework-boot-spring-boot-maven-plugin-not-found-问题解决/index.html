Plugin org.springframework.boot:spring-boot-maven-plugin: not found 问题解决

问题描述

IDEA 报了一个 Plugin org.springframework.boot:spring-boot-maven-plugin: not found 问题，

解决方法

在报红的地方加上版本描述：

这个具体的版本号可以看这个 pom.xml 文件的 spring-boot 版本信息。

然后刷新一下 maven 即可。

问题原因

出现这个问题的原因，我猜测是当前项目所用 spring boot 的版本和 maven 仓库中的插件版本不匹配，所有没有办法使用默认的版本号，比如，我这个项目用的 spring boot 版本是 2.5.5，而之前用的版本是 2.3.3，所以本地 maven 仓库中只有 2.3.3 版本的插件，这就导致了 2.3.3 和 2.5.5 版本的项目都不会默认匹配本地仓库中的插件，而必须明确给出版本号的说明。
